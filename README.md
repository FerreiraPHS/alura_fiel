# alura_fiel
App web simples para celebrar os times históricos do Corinthians: tela de entrada temática, busca por ano ou competição, cards com títulos, escalação e links para saber mais. Dados em JSON, interface em dark mode com foco em acessibilidade e responsividade, homenagem à Fiel no rodapé. Dica de Senha: Corinthians.
